# External Libraries

This directory contains third-party libraries that are bundled with this plugin.

## Bundled Libraries

The following libraries are included locally in this plugin:

### Chart.js Libraries
- `chart.umd.min.js`: Chart.js 4.5.1 (MIT License)
- `chartjs-adapter-date-fns.js`: Chart.js date-fns adapter (MIT License)

### jQuery UI
- `jquery-ui.css`: jQuery UI theme CSS 1.13.2 (MIT License)

### SheetJS
- `xlsx.full.min.js`: SheetJS Community Edition (Apache-2.0 License)

### Images
- `magic-emoji.png`: Twitter Emoji asset

## Note

All libraries are bundled locally to comply with WordPress.org plugin directory guidelines. No external resources are downloaded at runtime.
